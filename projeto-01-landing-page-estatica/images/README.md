# Imagens do Projeto

## Assets Necessários

### Imagens Principais
- `logo.svg` - Logo da empresa (120x40px)
- `hero-bg.webp` - Background do hero (1920x1080px)
- `dashboard-preview.webp` - Preview do dashboard (600x400px)
- `og-image.jpg` - Imagem para Open Graph (1200x630px)

### Logos de Clientes
- `client-1.svg` - Logo cliente 1 (120x60px)
- `client-2.svg` - Logo cliente 2 (120x60px)
- `client-3.svg` - Logo cliente 3 (120x60px)
- `client-4.svg` - Logo cliente 4 (120x60px)

### Outros
- `favicon.ico` - Ícone do site (32x32px)

## Otimizações Recomendadas
- Usar WebP para imagens fotográficas
- Usar SVG para logos e ícones
- Comprimir imagens com ferramentas como TinyPNG
- Implementar lazy loading para imagens below-the-fold
- Usar responsive images com srcset

## Ferramentas Sugeridas
- [Squoosh](https://squoosh.app/) - Compressão de imagens
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - Otimização de SVG
- [Unsplash](https://unsplash.com/) - Imagens gratuitas de alta qualidade
