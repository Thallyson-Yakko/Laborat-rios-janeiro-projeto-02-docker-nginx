# Projeto 1: Landing Page Estática de Alta Performance

## Problema
Uma startup precisa de uma landing page ultra-rápida para capturar leads. A página deve:
- Carregar em menos de 2 segundos globalmente
- Suportar picos de tráfego de 10.000 usuários simultâneos
- Ter SSL/TLS configurado
- Custar menos de $5/mês

## Desafio Técnico
Implementar uma solução usando S3 + CloudFront que atenda aos requisitos de performance e custo.

## Assets Fornecidos
- `index.html` - Landing page responsiva
- `css/style.css` - Estilos otimizados
- `js/app.js` - JavaScript minificado
- `images/` - Imagens otimizadas para web
- `favicon.ico` - Ícone do site

## Objetivos de Aprendizado
- Configurar S3 com objetos estáticos
- Implementar CloudFront para CDN global
- Configurar SSL com Certificate Manager
- Otimizar cache headers
- Implementar redirects HTTPS
- Configurar domínio customizado
